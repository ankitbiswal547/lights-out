Game Instructions

  • 1. As the name suggests("Lights out"), you have to turn all the lights off.
  • 2. Initially some of the lights will be turned 'ON'.
  • 3. On clicking any box with lights 'ON' will turn it 'OFF' and vice versa.
  • 4. But clicking on any box will toggle the adjacent boxes, so if any adjacent(top, bottom, left, right) box if 'ON' then it will be turned 'OFF' and vice versa.
  • 5. Try to win the game with least no. of moves.
  • 6. Hint: Try solving from top row to bottom row first :), so that the board is left with lights 'ON' boxes at the last row.
  • 7. PS : Every time u refresh the page you have to register again, so try not to.
